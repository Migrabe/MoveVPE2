# AGENTS.md - VPE Prompt Builder Context

## Project Summary
A sophisticated prompt management and construction tool for image generation models. Supports versioning, secure routing, and complex templates.

## Critical Directory Map
- `/server`: Core backend logic and routing.
- `/public`: Frontend assets (Vanilla JS).
- `/brain`: Project memory and agent tasks.
- `.agents/skills`: Local AI capabilities.

## Execution Rules
- Always use `npm run dev` to start the server.
- Default Port: 3000.
- Check `server/version_routing.js` before adding new endpoints.
